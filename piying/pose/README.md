# Pose Demo — 鼠标拖动骨骼并保存姿态

该 demo 用于在绑定过程中以鼠标拖动方式调整实体骨骼的姿态，并在按键时将当前姿态保存到 `pose/` 目录。

## 运行

```
python pose/demo.py
```

依赖：使用与现有项目相同的 `pygame` 环境（已在 `bone/requirements.txt` 中）。

## 操作

- 左键点击关节选择部件；按住左键拖动可旋转该部件（围绕该关节）。
- `Tab`：切换选中部件。
- `R`：重置为默认姿态。
- `S`：保存当前姿态到 `pose/pose_<时间戳>.json`。
- `Esc`：退出。

## 保存内容

保存文件包含：
- `origin`：当前屏幕上的骨骼中心位置（用于复现显示位置）。
- `angles`：九个部件的角度字典（单位：度）。
- `timestamp`：保存时间。

该 demo 仅绘制骨骼参考线，不改动任何现有文件；保存文件位于 `pose/` 目录下。
